Mobile Legends Bang Bang. Game Bertipe MOBA yang berani saingi Game PC?
By Jackmizan, a.k.a. Mizan Nur Qalam.

2016 adalah tahun dimana Game ini dirilis di Indonesia. Game besutan Moonton dari Negeri Tirai Bambu ini langsung menyedot perhatian para gamers di Indonesia. Sejak dirilis, Game ini sudah diunduh sebanyak 100 juta lebih di playstore. Hal ini membuktikan, bahwa komunitas pecinta game ini tidak bisa diremehkan. Bahkan Moonton selaku Developer Game ini mampu mengadakan Turnamen yang dinamakan MPL atau MLBB Professional League. Tak hanya sampai disitu, Mobile Legends juga resmi menjadi cabang olahraga e-sport yang di pertandingkan di SEA games 2019.

Game ini merupakan Game bertipe MOBA dimana dua tim masing-masing beranggotakan 5 hero akan bertarung dalam sebuah arena. Game ini menuntut para player untuk fokus, cermat dalam memilih strategi, dan kombinasi hero yang akan dipakai. Semakin bagus kerjasama dalam tim, maka kemenangan akan mudah didapatkan.





Alasan lain kenapa game ini menjadi viral di masyarakat adalah karena Game ini berplatform di smartphone yang notabene hampir semua lapisan masyarakat memilikinya. Berbeda dengan game di PC yang memiliki jumlah pengguna yang lebih sedikit. Game ini juga dapat dimainkan bersama teman anda secara online.

Game ini juga kami nilai sebagai Game dengan perkembangan game yang cepat, dimana inovasi dalam permainan yang dapat memanjakan user terus dilakukan oleh Moonton selaku developernya. Mulai dari sistem tier rank, jumlah hero/karakter yang terus diperbanyak, mode dalam permainan yang bervariasi, dan masih banyak lagi kejutan di dalam Game ini.

Sebagai penutup, kami tim dari Gadgetout berpendapat, bahwa Mobile Legends Bang Bang merupakan Game dengan pertumbuhan tercepat di Indonesia. Dalam kurun waktu 4 tahun, Game ini berhasil bercokol sebagai salah satu Game dengan User terbanyak di Indonesia.
